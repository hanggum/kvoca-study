# 🎓 어휘나라 - 초등학생 어휘 학습 프로그램

초등학생(1-6학년)을 위한 재미있고 효과적인 웹 기반 어휘 학습 프로그램입니다.

## ✨ 주요 기능

### 👨‍🎓 학생 포털
- **플래시카드 학습 (1단계)**: 인터랙티브한 카드 애니메이션으로 어휘, 뜻, 한자 학습
- **타이핑 연습 (2단계)**: 실시간 피드백으로 철자 정확성 향상
- **문장 만들기 (3단계)**: 창의적인 문장 작성 및 피드백
- **진도 추적**: 개인별 학습 현황 및 성취도 확인
- **학습 히스토리**: 과거 학습 기록 조회

### 👨‍🏫 선생님 포털
- **학생 관리**: 학생 계정 생성, 수정, 삭제
- **진도 모니터링**: 학생별 상세 학습 현황 확인
- **어휘 관리**: 어휘 데이터베이스 관리 (검색, 추가, 수정)
- **통계 대시보드**: 전체 학급 학습 통계 확인

## 🎯 학습 데이터

- **총 어휘 수**: 현재 18개 (1000개로 확장 가능)
- **학년별 분류**: 1-2학년 (5개), 3-4학년 (5개), 5-6학년 (8개)
- **각 어휘 포함 내용**:
  - 단어
  - 한자
  - 의미
  - 10개의 예문
  - 카테고리 (일상생활, 관계, 감정, 태도, 능력 등)

## 🏗️ 프로젝트 구조

```
vocab-learning-app/
├── index.html              # 메인 랜딩 페이지
├── css/
│   └── style.css          # 전체 디자인 시스템
├── js/
│   ├── auth.js            # 인증 및 사용자 관리 시스템
│   └── vocab-data.js      # 어휘 데이터베이스
├── student/
│   ├── login.html         # 학생 로그인
│   ├── dashboard.html     # 학생 대시보드
│   ├── flashcards.html    # 플래시카드 학습
│   ├── typing.html        # 타이핑 연습 (준비중)
│   └── writing.html       # 문장 만들기 (준비중)
└── teacher/
    ├── login.html         # 선생님 로그인
    └── dashboard.html     # 선생님 대시보드
```

## 🚀 실행 방법

1. **준비 사항**: 모던 웹 브라우저 (Chrome, Firefox, Safari, Edge)

2. **실행**:
   - `index.html` 파일을 웹 브라우저로 열기
   - 또는 로컬 서버 실행:
     ```bash
     # Python 3가 설치된 경우
     python3 -m http.server 8000
     # 브라우저에서 http://localhost:8000 접속
     ```

3. **로그인**:

   **학생 계정:**
   - 아이디: `student1`, `student2`, `student3`
   - 비밀번호: `1234`

   **선생님 계정:**
   - 아이디: `teacher`
   - 비밀번호: `teacher123`

## 💻 기술 스택

## 📚 어휘 데이터 확장

어휘를 추가하려면 `js/vocab-data.js` 파일의 `getSampleVocabulary()` 함수에 다음 형식으로 추가:

```javascript
{
    id: 'vocab19',
    word: '단어',
    hanja: '漢字',
    meaning: '뜻',
    grade: 3,  // 1-6
    category: '카테고리',
    examples: [
        '예문 1',
        '예문 2',
        // ... 10개의 예문
    ]
}
```
